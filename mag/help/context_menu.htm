Context Menu

Right-clicking anywhere on the magnifier window frame opens the context menu. The menu provides quick access to the main MAG features.

Item Description
Follow Mouse Toggles automatic cursor tracking. When enabled, the captured area is centered on the cursor while the magnifier window stays in place. A check mark appears next to the item while tracking is active.

While tracking, the window remains interactive. Press Space to stop tracking without opening the menu.
Settings Opens the settings dialog for graphics and capture backend selection. Available capture backends include GDI BitBlt, DXGI Desktop Duplication, Windows Graphics Capture, DWM Thumbnail, and DWM Private Visual.
Help Opens this help document.
Exit Closes MAG.
Tip: You can open the context menu while Follow Mouse is active by right-clicking the window frame.
