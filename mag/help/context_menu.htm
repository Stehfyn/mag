Context Menu

Right-clicking anywhere on the magnifier window frame opens the context menu. The menu provides quick access to the main MAG features.

Item Description
Follow Mouse Toggles automatic cursor tracking. When enabled, the magnifier window moves continuously to keep the cursor centred in the view. A check mark appears next to the item while tracking is active.

While tracking, the window is transparent to mouse input so it does not interfere with normal use. Press Space to stop tracking without opening the menu.
Help Opens this help document.
Exit Closes MAG.
Tip: To open the context menu while Follow Mouse is active, press Space first to pause tracking, then right-click the window frame.
