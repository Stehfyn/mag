Context Menu

Right-clicking anywhere on the magnifier window frame opens the context menu. The menu provides quick access to the view mode, settings, help, and exit commands.

Item Description
Window Uses the magnifier window position as the capture anchor. At 1.0x scale, the captured image lines up with the desktop area directly behind the window.
Follow Mouse Centers the captured area on the cursor while the magnifier window stays in place. The capture is clipped to the current capture bounds and still fills the window without stretching the texture. While tracking, the window remains interactive. Press Space to return to Window mode.
Lens Moves the magnifier as a cursor-following lens. The lens is topmost, non-activating, and click-through so input goes to the window underneath. Overlay controls are disabled in this mode.
Settings Opens the settings dialog. The current graphics backend is OpenGL. Capture backend choices are GDI BitBlt, DXGI Desktop Duplication, Windows Graphics Capture, DWM Thumbnail, and DWM Private Visual. The dialog also contains the Mouse-relative wheel zoom option, which is disabled by default.
Help Opens this help document.
Exit Closes MAG.
Tip: Window, Follow Mouse, and Lens are mutually exclusive. Selecting one turns off the other two.

Capture Backends

Backend Notes
GDI BitBlt Baseline desktop capture path.
DXGI Desktop Duplication Direct3D desktop duplication path with multi-monitor capture uploaded into one OpenGL texture.
Windows Graphics Capture Windows Graphics Capture based monitor capture path.
DWM Thumbnail DWM thumbnail based capture path.
DWM Private Visual Private DWM visual capture path.
