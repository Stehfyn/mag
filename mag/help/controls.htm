Controls & Keyboard Shortcuts

Mouse Controls

Action Effect
Mouse Wheel Zoom in or out. The zoom level adjusts with a non-linear curve for fine control at high magnification.
Left-click and drag Move the magnifier window to a new position on screen.
Drag any edge or corner Resize the magnifier window. The captured region scales with the window size.
Right-click Open the context menu. See Context Menu for details.

Keyboard Shortcuts

Key Effect
Space Stop Follow Mouse mode. The window remains in place at the last tracked position. Only active while tracking is enabled.
Escape Close MAG. Only active when the magnifier window has focus.
Note: When Follow Mouse is active the magnifier window is transparent to mouse input. To interact with the window directly (drag, resize, or right-click) you must first stop tracking by pressing Space, or by selecting Follow Mouse again from the context menu.

Zoom Behaviour

The zoom level ranges from minimum (the window shows an area close in size to itself) to maximum (each screen pixel fills a large portion of the window). The zoom step size scales automatically so that coarse changes are fast when zoomed out and fine changes are precise when zoomed in.
