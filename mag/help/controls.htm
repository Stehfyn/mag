Controls & Keyboard Shortcuts

Mouse Controls

Action Effect
Mouse Wheel Zoom in or out in smaller, non-linear steps for fine control at high magnification.
Mouse Wheel with mouse-relative zoom enabled Zoom around the cursor position instead of the current view center. This setting is off by default and only applies outside tracking modes.
Drag the frame Move the magnifier window to a new position on screen.
Drag any edge or corner Resize the magnifier window. Rendering continues on vblank while the modal resize loop is active.
Left-click and drag minimap Pan the visible source rectangle inside the captured desktop area. The minimap is available while zoomed in, while dragging or resizing, and in the pinned 1.0x state after panning.
Right-click Open the context menu. See Context Menu for details.

Keyboard Shortcuts

Key Effect
Space Stop Follow Mouse mode and return to Window mode. Only active outside Lens mode.
Escape Close MAG. Only active when the magnifier window has focus.
Note: In Follow Mouse mode the window remains interactive. Clicking the minimap pans the view without leaving follow mode. In Lens mode, the window is click-through and the overlay UI is disabled.

Zoom Behaviour

The zoom level ranges from minimum (the window shows an area close in size to itself) to maximum (each screen pixel fills a large portion of the window). The zoom step size scales automatically so that coarse changes are fast when zoomed out and fine changes are precise when zoomed in. At 1.0x scale, panning with the minimap pins the source view until the actual window is moved.

Minimap Overlay

The minimap shows the capture area, monitor boundaries, the current visible source rectangle, and the window rectangle. It fades like video playback controls after mouse activity stops, but remains visible while dragging or resizing when a zoomed or pinned view needs it.
