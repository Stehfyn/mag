MAG Screen Magnifier

MAG is a lightweight, high-performance screen magnifier for Windows. It renders a real-time zoomed view using OpenGL, with vblank-paced updates, high-DPI support, and multi-monitor capture support.

Getting Started

When MAG launches, a resizable magnifier window appears on your desktop. The window shows an enlarged view of whatever is beneath it.

  • Scroll the mouse wheel over the window to zoom in or out.
  • Drag the window by its frame to position it anywhere on screen.
  • Right-click the window frame to open the context menu.
  • Open Settings to choose the capture backend and optional mouse-relative wheel zoom.
Tip: Use Window mode for a fixed magnifier, Follow Mouse to keep the window in place while centering capture on the cursor, or Lens for a click-through lens that follows the cursor.

View Modes

Mode Behavior
Window The capture is tied to the magnifier window position. At 1.0x scale, the view matches the desktop area behind the window.
Follow Mouse The window stays where it is, but the capture is centered on the cursor and clipped to the active capture bounds.
Lens The magnifier follows the cursor and remains click-through so pointer input reaches the window underneath.
